# Cuando las Letras, el patinaje artístico sobre hielo y el Frontend se conocieron

## L'Anima del Ghiaccio - Digital Magazine Concept
Una pieza de CSS Art, de ingeniería frontend y de editorial inspirada en el patinaje artístico y la estética olímpica de Milano-Cortina 2026.

![Portada](./assets/Captura.png) 

## El Concepto - mi experiencia
Como Licenciada en Letras Clásicas, colaboré un año en una editorial de la UNAM, donde aprendí como crear libros y revistas académicas. Gracias a esta fromación y mi pasión como patinadora sobre hielo retirada, esto junto con mis nuevas habilidades como Frontend Dveloper y Diseñaodara UX he creado una mini muestra de la portada de una revista:

* Filología Digital: Uso de lemas en latín (Ad Astra, Citius Altius Fortius) como elementos de UX.

* Diseño Editorial: Traslación de layouts de revistas físicas a CSS Grid dinámico.

## Desafíos Técnicos Superados 
* Grid vs. Absolute: La estrategia de liberar a los patinadores del flujo del grid para permitir composiciones artísticas sin romper la estructura de las estrellas.

* Micro-interacciones: El efecto de brillo en la tipografía y el sistema de estrellas con box-shadow.

* Tipografía Responsiva: Implementación de writing-mode vertical para emular el diseño de revistas de lujo.

## Stack Tecnológico
* HTML5 (Semántico)

* CSS3 (Custom Properties, Grid, 3D Transforms)

* FIGMA

## 🚀 Próximos pasos - The Roadmap
Ahora que está lista la portada planeo continuar con la formación de la revista, utilizaré mis habilidades tanto de dev como de escritora y excoach de patinaje. Técnicamente me enfocaré en convertir esta portada en un Códice Interactivo:

* The Spine (La Bisagra): Perfeccionaremos el eje de rotación lateral para que la hoja no solo gire, sino que tenga esa "curvatura" orgánica del papel.

* Layout de Artículos: Crearemos plantillas de artículos internos usando CSS Multi-column o sub-grids para que parezcan reportajes reales.

* Ad Astra Experience: Haremos que el clic en la esquina doblada sea el disparador de toda la transición 3D.
